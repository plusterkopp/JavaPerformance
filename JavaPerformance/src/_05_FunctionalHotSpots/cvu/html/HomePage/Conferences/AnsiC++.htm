ABSTRACT
 
    This lecture gives an overview of features added to the language during the standardization process. ANSI C++ now supports: 
    • member templates,
    • partial template specialization,
    • explicit function template argument qualification,
    • default template arguments,
    • template template arguments,
    • function try blocks,
    • runtime type information,
    • namespaces,
    • new-style casts,
    • various new keywords (e.g. typename, mutable, explicit),
    • and many more interesting features.
