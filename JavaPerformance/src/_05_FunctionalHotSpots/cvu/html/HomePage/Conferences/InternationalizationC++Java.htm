ABSTRACT
 
Internationalization is the process of building into software the potential for worldwide use, a requirement of increasing importance.  Both C++ and Java support this process by means of so-called locales. The lecture will give an introduction to locale objects in the ISO/ANSI Standard C++ Library.  We will demonstrate the use of standard C++ locales in terms of an example: "multicultural stream I/O" using locales in conjunctions with the new standard IOStreams. We will then explore Java locales and localized resources in order to show correspondences and point out differences from the internationalization support in C++.
