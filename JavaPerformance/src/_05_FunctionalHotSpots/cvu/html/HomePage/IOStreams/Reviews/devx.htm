Book Review of the week 
(found at www.devx.com ) 

 
 
 
Standard C++ IOStreams and Locales: Advanced Programmer's Guide and Reference 
By Angelika Langer et al. 
Reading,MA:
Addison-Wesley 
ISBN 0201183951 
672 pages 
Price: $49.95
(Reviewed 8/25/00)
 
 
Island in the C++ Streams 
This comprehensive reference guide to the standard C++ stream and locale frameworks includes excellent documentation for beginners and experienced users alike
 
Amidst a torrent of new C++ books that discuss the standard IOStream Library, Standard C++ IOStreams and Locales: Advanced Programmer's Guide and Reference by Angelika Langer and Klaus Kreft is unique in two respects: It's among the few books that explain how to extend the standard stream classes, and it's currently the only book that provides a comprehensive account of the standard locales framework of C++. Bjarne Stroustrup recently added a brief online introduction (in PDF format) to C++ locales as an appendix to his book C++ Programming Language, Special Edition . 

Langer's and Kreft's book contains three parts: stream input and output, internationalization, and a reference guide. Although you might be tempted to skip the first part, as it ostensibly discusses the basics of the IOStream framework, my advice is to read it thoroughly. Most likely, even experienced users can learn a thing or two from the discussion and improve their skills and understanding of the IOStream classes. For instance, many programmers use the endl manipulator abundantly to force stream flushing, although in many cases this practice is either redundant or incurs unnecessary performance overhead. 

Remember that the C++ IOStream framework underwent substantial modifications and adjustments throughout the standardization process. For example, char-oriented streams were templatized to enable wide character support, the entire C++ Standard Library was moved into its dedicated ::std namespace instead of being declared in the global namespace, parts of the original IOStream classes were deprecated and replaced by newer components, the names of the standard headers were changed, and so on. Therefore, beginners and accomplished programmers alike can benefit from reading this up-to-date account of the IOStream classes. 

The second part is dedicated to internationalization. Sadly, very few books have addressed this hefty topic before, in spite of the fact that internationalization and localization play a major role in almost every nontrivial C++ project these days. This is a classic chicken-and-egg problem. Few C++ books address internationalization. Consequently, few programmers use the C++ <locales> framework. Because of this, the demand for books on internationalization isn't high. Langer and Kreft are pioneers in this regard. They dedicate a significant part of their book to discussing internationalization from the ground up using the state-of-the-art <locales> framework. This part opens with a brief introduction that explains the difference between internationalization and localization, and demonstrates some of the cultural differences among locales—for instance, date and time formats, monetary systems, numeric notation, and, of course, languages and codesets. 

Next, Langer and Kreft portray the two fundamental units of the standard <locales> framework, namely locales and facets. Finally, advanced techniques such as direct use of a facet through a stream, the <locales> class hierarchy, and memory management are explored. Although your familiarity with the concepts is not assumed, the discussion might be too advanced for programmers with no prior experience in software internationalization. Therefore, I recommend that beginners read Stroustrup's online introduction (cited above) first. 

The final part serves as a reference guide. It contains a detailed description of the standard IOStream and <locales> frameworks, presenting a description and synopsis of each header file and its constituents. The book contains seven appendices, some of which are highly useful: e.g., "Differences Between Classic and Standard IOStreams," "The Relationship Between C and C++ Locales," and "New C++ Features and Idioms." 

Overall, Standard C++ IOStreams and Locales: Advanced Programmer's Guide and Reference is a comprehensive, up-to-date tutorial and reference guide for the C++ IOStream and <locales> framework. Programmers who wish to broaden their proficiency in these areas will certainly find the book beneficial.
