REVIEW International Programming for Microsoft Windows
Author: David A. Schmitt 
Microsoft Press , 2000 
ISBN: 1-57231-956-9
Gives an excellent introduction to the problem of international programming and a lucid overview of all relevant APIs for internationalization of C and C++ programs under Windows. Not only does it provide sufficient detail, but it's up-do-date as well. 
DETAILS Standard C++ IOStreams and Locales
Authors: Angelika Langer & Klaus Kreft 
Addison-Wesley , January 2000 
ISBN 0-201-18395-1 
 Provides detailed coverage of international features in standard C++*1).
 
*1)Another source of information about the standard C++ locales is an appendix that Bjarne Stroustrup added to his book C++ Programming Language, Special Edition . The appendix is available online (in PDF format).
Developing International Software for Windows 95 and Windows Nt
Author: Nadine Kano 
Microsoft Press , 1995 
ISBN: 1-55615-840-8
Contains extensive detail about code pages, Unicode, and other aspects of Win32. Has not been updated vor Windows 98 and Windows NT 4.0 and Windows 2000, but most of the material is still valid.
Understanding Japanese Information Processing
Author: Ken Lunde 
O'Reilly, 1993 
ISBN: 1-56592-043-0
Gives insight into encoding of Japanese character sets and beyond that aids understanding of multibyte character encodings in general.
