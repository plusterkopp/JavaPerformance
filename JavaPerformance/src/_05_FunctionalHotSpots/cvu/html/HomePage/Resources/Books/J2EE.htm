EJB - Enterprise Java Beans
REVIEW Mastering Enterprise Java Beans, 2nd Ed.
Ed Roman, Scott W. Ambler, Tyler Jewell, Floyd Marinescu
John Wiley & Sons, December 2001
ISBN: 0471417114
An excellent introduction to EJB with several appendices providing introductions into related topics such as JNDI, RMI, XML, etc.
EJB Design Patterns
Floyd Marinescu
John Wiley & Sons, February  2002
ISBN: 0471417114
A wonderful companion to an introductory EJB book which covers best practice programming tips.
REVIEW Enterprise JavaBeans, 3rd Ed.
Richard Monson-Haefel
O'Reilly, October 2001
ISBN: 0596002262
Another excellent introduction to EJB.  A little more thorough than Ed Roman's book and with a stronger focus on EJB alone; it does not have any of the appendices that "Mastering EJB" has on related topics.
WebSphere 4.0 AEs Workbook for Enterprise JavaBeans, 3rd Ed.
Kyle Brown
O'Reilly & Associates, September 2002
ISBN: 0596004184
WebLogic Server 6.1 Workbook for Enterprise JavaBeans, 3rd Ed.
Greg Nyberg
O'Reilly & Associates, September 2002
ISBN: 0596004176
The companion workbooks for Monson-Haefel's book on EJB.

JDO - Java Data Objects
Java Data Objects
Robin M. Roos
Bk&CD-Rom, October 2002
ISBN: 0321123808
Moderately recommendabl.  Currently (end of 2002) it is the only title available on JDO.  It does not provide much information beyond the JDO specification.
