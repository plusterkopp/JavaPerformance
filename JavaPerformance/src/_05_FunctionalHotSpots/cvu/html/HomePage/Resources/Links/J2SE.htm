Below is a directory of websites related to the Java programming language in general and J2SE (Java Standard Edition) in particular.
 

Directories

Java at Sun Microsystems
The source for Java technology at Sun Microsystems, Inc.. Hosts download of products, API specification, tutorials, and everything Sun has to say about Java. The current version 5.0 of the Java Standard Edition (J2SE) is available for download from  http://java.sun.com/j2se/1.5.0/download.jsp. 

ACCU
Links to various Java resources are maintained by the Asscociation of C and C++ Users (ACCU) in the UK.  See ACCU Java Resources for more information.

Articles

Brian Goetz
Brian Goetz writes a column entitles "Java Theory and Practice" published at the IBM developerWorks website with numerous articles related to Java that are worth reading.

Mark Davis
Mark Davis is author of column named " Durable Java " that appeared in Java Report.  His articles covered core Java topics and pointed out traps and pitfalls in the allegedly "easy" programming language Java and are a must-read for every Java programmer. Drafts of his articles are available on his website.

Klaus Kreft & Angelika Langer
We (Klaus Kreft and Angelika Langer) wrote a couple of similar articles covering the same or related ground.  Our articles appeared in German in JavaSpektrum and in English in JavaSolutions.

Roedy Green
A website with a Java glossary defining acronyms and terms seen in comp.lang.java.*. In addition there are various tutorials and a page with numerous gotchas.

Books

Go to BOOKS J2SE for books on Java programming.
