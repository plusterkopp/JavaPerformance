I work as an independent freelance trainer, and develop and conduct courses in Europe and North America. 

My areas of expertise include advanced C++ and Java programming, concurrent
programming and Enterprise Java.

I am co-author with Klaus Kreft of the book "C++ Standard IOStreams
and Locales" (Addison Wesley, 2000). I wrote the column "Effective Standard Library" for the US magazines C++ Report and C/C++ Users Journal. Currently I write a column named "Effective Java" for the German magazine Java Spektrum. 

I enjoy speaking at conferences all over the world. I am an observing member of the ISO/ANSI C++ standards committee and the Java Community Process JCP.
